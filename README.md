# Age Calculator

This is a simple **Age Calculator** application that calculates the user's age based on their date of birth. Built using HTML, CSS, and JavaScript, it provides a user-friendly interface for age calculation.

---

## Features

- **Date Input**: Users can input their date of birth.
- **Age Calculation**: Displays the user's age in years, months, and days.
- **Responsive Design**: Works well on all devices.

---

## Technologies Used

- **HTML**: Structure of the webpage.
- **CSS**: Styling for the application.
- **JavaScript**: Logic for age calculation.
